¿Que es Plc Node?

PLC NODE es un modulo análogo a al funcionamiento de un controlador lógico programable que se considera como un nodo donde se concentra información, se controla y optimiza las acciones de un proceso industrial por medio de la implementación de internet y está orientado al procesamiento de entradas y salidas donde en las entradas se utilizan sensores y en las salidas se utilizan actuadores. El proyecto cuenta con un software emergente que implementa tecnología de internet de las cosas. El software es un entorno de programación integrado que es capaz de implementar instrucciones a bloques, donde cada bloque tiene funcionalidades diferentes, dando la capacidad a PLC NODE de conectar servicios externos, creación de interfaces de usuario, conexión a una gran variedad de protocolos de comunicación, procesamiento de señales y algunos módulos de inteligencia artificial.

A nivel global, los controladores lógicos programables son demandados, debido a que se necesitan procesos automatizados para eficientar la productividad y generación de productos en masa, con la llegada de cuarta revolución industrial cada vez la industria se encuentra con mayor competencia entre sí y con ello la demanda de estos productos Plc´s de ultima generación con internet son demandados para la implementación de concepto de internet de las cosas en el área industrial.



Caracteristicas técnicas

Microcontrolador IOT
Alimentación directa a 110VAC.
Entradas/Salidas
    6 entradas analogas adaptadas a 24 VDC
    6 salidas adaptadas a 24 VDC
      4 salidas digitales
      2 salidas analogas.
Comunicación.
   Alámbrica.
      Protocolo Usb
   Inalámbrica
      Protocolo Wifi en conjunto con Websockets.

Node-Red supports

Node-RED es una herramienta de desarrollo basada en flujo para programación visual desarrollada originalmente por IBM para conectar dispositivos de hardware, API y servicios en línea como parte del Internet de las cosas. Node-RED proporciona un editor de flujo basado en navegador web, que se puede utilizar para crear funciones de JavaScript. Los elementos de las aplicaciones se pueden guardar o compartir para su reutilización.
